# 🎵 Pitch Climber

Climb a cliff with your voice. Higher pitch = higher altitude. Hold specific notes to lock onto ledges. Falsetto required by level 3.

▶️ [Play Now](https://ashleywolf.github.io/pitch-climber/)

## How to Play

Open in Chrome and allow microphone access. Hum, sing, or shriek to climb. Hold notes at target frequencies to grab ledges. 3 levels with increasing difficulty.

## Tech

Web Audio API with autocorrelation pitch detection. HTML5 Canvas. No external dependencies, single file, zero build step.

## Input

Microphone (voice pitch via AnalyserNode)
